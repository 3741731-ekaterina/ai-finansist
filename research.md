### Критические слабые места ❌

#### 1. Нет социального доказательства в первом экране
Отзывы, количество студентов, результаты выпускников — всё это скрыто или отсутствует. Первое что видит посетитель — только ваши слова без подтверждения.
→ **Решение:** Добавить под Hero строку «100+ финансистов уже прошли курс · ★★★★★ 4.9»

#### 2. Скрытые цены
Стоимость курса находится в разделе «Тарифы» — нужно скроллить. Аудитория финансистов привыкла к прозрачности. Скрытая цена = сигнал «будут торговаться».
→ **Решение:** Добавить «от X ₽» в Hero или в строку доверия.

#### 3. Нет конкретного результата в заголовке H1
«НЕЙРОСЕТИ для финансиста» — это тема, не результат. Что конкретно получит финансист после курса?
→ **Решение:** Переписать H1 под формат: *«[Результат] за [срок] для [аудитория]»*

#### 4. «Зачем вам этот курс?» — слабая секция
5 абстрактных тезисов про «рост требований рынка» не создают личного резонанса. Это не боль клиента — это отраслевая аналитика.
→ **Решение:** Заменить на конкретные трансформации: «Было / Стало» или конкретные кейсы «Сэкономила 8 часов в неделю на отчётности».

#### 5. Нет программы курса в видимой зоне
6 недель — но что именно изучается? Структура модулей должна быть видна без поиска.
→ **Решение:** Добавить раскрываемый блок «Программа» после Hero.

#### 6. Нет дедлайна / срочности
Ничто не мотивирует записаться СЕЙЧАС. Финансовая аудитория откладывает решения «на потом» — нужен триггер.
→ **Решение:** «Старт 15 апреля · Осталось X мест» (прямо в Hero).

#### 7. Две конкурирующие кнопки без иерархии
«ЗАНЯТЬ МЕСТО» и «записаться на обучение» — делают одно и то же. Снижает конверсию.
→ **Решение:** Primary CTA: «Занять место →» + Secondary: «Узнать подробнее ↓»

## 3. ИДЕАЛЬНАЯ СТРУКТУРА ЛЕНДИНГА КУРСА

Логика (модель **AIDA+**): **Attention → Interest → Desire → Action + Trust + Urgency**

Психологический путь покупателя: «что это?» → «это для меня?» → «это работает?» → «это безопасно?» → «хочу сейчас»

```
БЛОК 1: HERO          → ЗАХВАТ (Attention)    — "Что это? Это для меня?"
БЛОК 2: ПРОБЛЕМА      → БОЛЬ (Pain)           — "Да, это про меня!"
БЛОК 3: РЕЗУЛЬТАТЫ    → НАДЕЖДА (Hope)        — "Вот что я получу"
БЛОК 4: ПРОГРАММА     → ЦЕННОСТЬ (Value)      — "Вот что я изучу конкретно"
БЛОК 5: ДЛЯ КОГО      → ИДЕНТИФИКАЦИЯ (Fit)   — "Это точно для таких как я"
БЛОК 6: АВТОР         → ДОВЕРИЕ (Authority)   — "Этому человеку можно доверять"
БЛОК 7: ОТЗЫВЫ        → УВЕРЕННОСТЬ (Proof)   — "Это уже работает у других"
БЛОК 8: ТАРИФЫ        → РЕШЕНИЕ (Decision)    — "Мне это по силам финансово"
БЛОК 9: FAQ           → СНЯТИЕ БАРЬЕРОВ       — "Мои сомнения развеяны"
БЛОК 10: ФИНАЛ CTA    → ДЕЙСТВИЕ (Action)     — "Покупаю сейчас"
```

### Блок 0. НАВИГАЦИЯ (sticky)

```
Логотип/имя курса  |  Программа  Автор  Отзывы  Стоимость  |  [Занять место →]
```

**Важно:** кнопка CTA в навбаре видна всегда при скролле.

### Блок 1. HERO 🔴 Критически важен

**Цель:** за 3 секунды объяснить: что это, для кого, какой результат, почему сейчас.

**Состав:**
- Тег над заголовком: «✦ 6-недельный онлайн-курс · Старт 15 апреля»
- **H1** — конкретный результат для финансиста
- Подзаголовок — уточнение аудитории + формат
- Два CTA: Primary «Занять место →» + Secondary «Смотреть программу ↓»
- Строка доверия под кнопками: «★★★★★ · 120 выпускников · Старт 15 апреля · 14 мест осталось»
- Визуал: фото автора или скриншот рабочего инструмента (не абстрактные иконки)

**Примеры H1 (от слабого к сильному):**

❌ «Нейросети для финансиста» — тема, не результат

⚠️ «Научитесь применять нейросети в финансовой работе» — что делать, но зачем?

✅ «Сократите рутину на 8 часов в неделю — с помощью нейросетей в финансах»

✅ «Финансисты, которые используют AI, зарабатывают на 30% больше. Стань одним из них.»

✅ «6 недель — и вы автоматизируете то, на что раньше уходили дни»

```
Ручная сверка данных         →   Автоматическая проверка
Не знаете как использовать   →   10+ готовых промптов для
ChatGPT в работе                  финансовых задач
Страх замены AI              →   Конкурентное преимущество
```

### Блок 5. ПРОГРАММА КУРСА 🔴 Критически важен

**Цель:** снять страх «а вдруг не то, что нужно».

### 10 трендов для применения на курсовом лендинге

---

#### Тренд 1 — AI-Inspired Aesthetic ⭐⭐⭐⭐⭐
**Что это:** анимированные Aurora-градиенты, сетчатые mesh-фоны, неоновые glow-эффекты. Визуальный язык «передовых технологий».

**Применение для вас:**
- Aurora background в Hero-секции (анимированный градиент за заголовком)
- Glowing effect на CTA-кнопке
- Particle/dot grid pattern как подложка

**Сложность:** средняя (CSS @keyframes + blur)

**Код:**
```css
.hero-bg {
  background: radial-gradient(ellipse at 30% 50%, rgba(121,89,173,0.3) 0%, transparent 60%),
              radial-gradient(ellipse at 70% 20%, rgba(195,48,190,0.2) 0%, transparent 50%);
  animation: aurora 8s ease infinite;
}
@keyframes aurora {
  0%, 100% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
}
```

---

#### Тренд 2 — Bento Grid Layout ⭐⭐⭐⭐⭐
**Что это:** асимметричная сетка из разноразмерных карточек. Apple применяет с 2023, доминирует в 2025–2026.

**Применение:** блок «Что вы получите» — карточки разного размера, крупная главная, мелкие вспомогательные.

**Сложность:** низкая (CSS Grid)

---

#### Тренд 3 — Scroll-triggered Animations ⭐⭐⭐⭐⭐
**Что это:** элементы появляются, трансформируются, накручиваются при скролле.

**Применение:**
- Счётчик «120+ выпускников» накручивается при входе в viewport
- Карточки модулей появляются последовательно при скролле
- Строка логотипов компаний бесконечно прокручивается (marquee)

**Сложность:** низкая–средняя (IntersectionObserver + CSS animation)

---

#### Тренд 4 — Glassmorphism 2.0 ⭐⭐⭐⭐
**Что это:** полупрозрачные карточки с blur-эффектом. У вас уже есть — нужно усилить.

**Применение:** карточки тарифов, отзывов, программы курса.

**Код:**
```css
.glass-card {
  background: rgba(255, 255, 255, 0.08);
  backdrop-filter: blur(16px);
  border: 1px solid rgba(255, 255, 255, 0.12);
  border-radius: 16px;
}
```

#### Тренд 8 — Микроинтерактивность ⭐⭐⭐⭐
**Что это:** мгновенная обратная связь на действия пользователя. Hover, focus, click — всё анимировано.

**Применение:**

### Портрет целевой аудитории

**Ядро ЦА:** финансисты, 28–50 лет.

#### Сегмент 1 — Финансовый директор / CFO
- Выручка компании: 100 млн+ руб.
- Боль: тратит время на сборку данных, хочет стратегический фокус
- Мотивация: эффективность, статус «передовой CFO»
- Возражение: «Не хватает времени на учёбу»
- **Триггер:** «CFO, которые используют AI, принимают решения на 40% быстрее»

#### Сегмент 2 — Финансовый менеджер / аналитик
- Уровень: middle/senior
- Боль: рутинные отчёты съедают рабочее время
- Мотивация: карьерный рост, повышение ценности на рынке труда
- Возражение: «Это слишком технически сложно»
- **Триггер:** «Курс создан не для программистов — для финансистов»

#### Сегмент 3 — Бухгалтер / главбух
- Боль: страх замены AI, не понимает как использовать
- Мотивация: безопасность, уверенность в будущем
- Возражение: «Я в этом не разберусь»
- **Триггер:** «AI не заменит финансиста — заменит того, кто не умеет работать с AI»

### Топ-5 формул заголовков для H1

**Формула 1 — Результат + срок:**
> «Сократите рутину на 8 часов в неделю за 6 недель»

**Формула 2 — Боль → Решение:**
> «Устали тратить дни на отчёты? Научитесь делать это за 20 минут с AI»

**Формула 3 — Конкурентное преимущество:**
> «Финансисты с AI-навыками зарабатывают на 30% больше. Вы готовы?»

**Формула 4 — Трансформация:**
> «Из финансиста в таблицах — в финансиста с суперсилой AI»

**Формула 5 — Страх + гарантия:**
> «AI не заменит вас — если вы знаете, как с ним работать»

### Триггеры доверия — рейтинг по силе для финансистов

**Уровень 1 (максимальный):**
1. **Логотипы компаний-работодателей** (Сбер, ВТБ, Deloitte, McKinsey, Газпром) — для финансистов, понимающих иерархию компаний
2. **Отзывы с должностью + компанией + цифрой** — «Мария Иванова, Financial Controller, Леруа Мерлен»
3. **Конкретные цифры результатов** — «с 3 дней до 4 часов», «40 часов экономии в месяц»

**Уровень 2 (высокий):**
4. **Личность эксперта-финансиста** — «CFO, который освоил AI» > «AI-тренер, объясняющий финансы»
5. **Программа с результатом каждого модуля** — «После урока 3 вы автоматизируете отчёт ДДС»
6. **Гарантия ROI** — «Если не сэкономите 5 ч/неделю — вернём деньги»

**Уровень 3 (дополнительный):**
7. Счётчик «1200+ финансовых специалистов» (с уточнением профессии)
8. Сертификат «добавьте в LinkedIn»
9. Медиаупоминания (Forbes, РБК, VC.ru, ACCA/CIMA)
10. Скриншоты реальных промптов и результатов из ChatGPT

### Описание программы — правильный формат

**Неправильно:**
> Модуль 2. Работа с ChatGPT — Основы промпт-инжиниринга — Практические задания

**Правильно:**
> **Модуль 2. Автоматизация финансовой отчётности**
> → Результат: пишете промпт, генерирующий управленческий отчёт из Excel за 10 минут
> — Промпты для анализа план/факт отклонений
> — Шаблоны ежемесячного пакета отчётности
> — Инструменты: ChatGPT-4o, Microsoft Copilot

**Правила:**
1. Каждый модуль = конкретный рабочий результат («После этого модуля вы сможете...»)
2. Называйте реальные задачи: «управленческая отчётность», «бюджет», «отчёт для совета директоров»
3. Указывайте конкретные инструменты (ChatGPT, Copilot, Claude)
4. Показывайте прогрессию от базового к продвинутому

### Core Web Vitals — цели
